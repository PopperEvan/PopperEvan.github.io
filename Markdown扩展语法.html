Markdown扩展语法

在上一篇文章,记录了Markdown的基本语法，本文记录下我们常用的扩展语法。

删除线

当需要表示废弃的文字时或过时的方法，就需要删除线。加删除线的文本，在编译为 Html 后会被<s>和</s>标签包围。
删除线的表现方法是：文字前后用~~包围。切记是半角~而非全角～，注意全角和半角的区别。

~~要删除的方法~~

效果：要删除的文字

单元格和表头

使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行：

姓名  | 代表作
---- | ---
苏轼  | 水调歌头
纳兰性德 | 饮水词
李太白 | 梦游天姥吟留别
姓名 代表作
苏轼 水调歌头
纳兰性德 饮水词
李太白 梦游天姥吟留别

为了美观，可以使用空格对齐不同行的单元格，并在左右两侧都使用 | 来标记单元格边界：

| 姓名     |        代表作 |
| ----    | ------------ |
| 苏轼     |      水调歌头 |
| 纳兰性德 |        饮水词 |
| 李太白   | 梦游天姥吟留别 |
姓名 代表作
苏轼 水调歌头
纳兰性德 饮水词
李太白 梦游天姥吟留别

对齐

在表头下方的分隔线标记中加入 :，即可标记下方单元格内容的对齐方式：

  • :--- 代表左对齐
  • :--: 代表居中对齐
  • ---: 代表右对齐
| left    |      center | right   |
| :---    |      :--:   | ---:    |
| 苏轼    |       李太白  | 纳兰性德 |
| 水调歌头 | 梦游天姥吟留别 |   饮水词 |
left center right
苏轼 李太白 纳兰性德
水调歌头 梦游天姥吟留别 饮水词

如果不使用对齐标记，单元格中的内容默认左对齐；表头单元格中的内容会一直居中对齐（不同的实现可能会有不同表现）。

锚

　锚是一种特殊的链接，专门用以标志出可供跳转到位置。 　

注意：由于标题可以折叠其下的内容，所以锚可能被它前面的标题折叠起来。此时锚是不起作用的。

示例：在指定位置添加：

<a id="jump">我是锚点</a>

然后此处添加：

[跳转锚点](#jump)
